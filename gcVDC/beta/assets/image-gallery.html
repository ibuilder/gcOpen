JavaScript Usage Examples

Get System Icon

const structuralIcon = gcVDC.getSystemIcon('Structural', true);
document.getElementById('system-icon-example').innerHTML = structuralIcon;

Get Placeholder Image

const placeholder = gcVDC.getPlaceholderImage('empty-state', 'No data available');
document.getElementById('placeholder-example').innerHTML = placeholder;

Get User Avatar

const avatar = gcVDC.getUserAvatar(null, true);
document.getElementById('avatar-example').innerHTML = avatar;
